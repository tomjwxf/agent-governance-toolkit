# Support

## How to file issues and get help

This project uses GitHub Issues to track bugs and feature requests. Please search the
[existing issues](https://github.com/microsoft/agent-governance-toolkit/issues) before filing
new issues to avoid duplicates. For new issues, file your bug or feature request as a new Issue.

For help and questions about using this project, please use:

- **[GitHub Discussions](https://github.com/microsoft/agent-governance-toolkit/discussions)** — Ask questions, share ideas, and discuss the project
- **[GitHub Issues](https://github.com/microsoft/agent-governance-toolkit/issues)** — Report bugs and request features
- **[Documentation](https://github.com/microsoft/agent-governance-toolkit/tree/main/docs)** — Read the project documentation

## Microsoft Support Policy

Support for this project is limited to the resources listed above.
